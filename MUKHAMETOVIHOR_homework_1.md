# Homework 1 - Linux Basics
 ## Task 1 Basic navigation
1.pwd - Current directory поточний каталог
2.Показати вміст домашнього коталогу
ls ~
пояснення командо ls показує список файлів і папок у домашньому католозі 
3.Перейти у католог Downloads і показати його вміст
cd ~/Downloads
ls
4.Створити порожній файл(без touch )
> test.txt
пояснення: створює порожній файл
5.Переглянути вміст файлу
cat test.txt
команда cat показує вміст файлу
6.Перейти в домашній каталог обсолютним шляхом
cd /home/ubuntu
пояснення: обсолютний шлях починається з /.
7.Перехід у домашній каталог відносним шляхом
cd ..
пояснення : перехід на рівень вище


  завдання 2 робота з документацією
виконані команди:
man ls
help cd
man cat
cp --help
mv --help
 
 відповіді
1.ключ ls для прихованих файлів: -a
2.ключ cat для нумерації рядків: -n
3.Різниця між man і --help:
man показує повну документацію, а --help коротку довідку
  Завданя 3 Міні сценарій
cd ~/Downloads

Пояснення:сценарій створює папку,файл,переходить між каталогами та відкриває документацію
